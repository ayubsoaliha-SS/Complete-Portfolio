# Soaliha Ayub Portfolio

Premium React/Vite portfolio for Soaliha Ayub, featuring a polished hero, project showcase, GitHub stats, contact links, and a downloadable resume.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content To Keep Updated

- Resume: `public/resume.pdf`
- Project links and details: `src/data/projects.js`
- Contact links: `src/components/Contact.jsx`
- Deployment URL: add a canonical URL and absolute Open Graph image URL in `index.html` after hosting
- Screenshots: swap concept covers for real product screenshots as each project goes live
